## 小根堆:
   new PriorityQueue<>((a, b) -> a - b)
   求 opK或者第几大或小问题,只保留K个大小,来一个扔掉最小堆顶元素
   
## 动态规划   
   连续最值问题,连续性动态规划 dp[i]=f(dp[i-1]),迭代元素获取
   
## 滑动窗口
   j-i=k nums[i]...nus[j]   
   
## 单调栈

## 数组问题
